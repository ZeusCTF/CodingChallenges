# Data Structures

Standalone implementations of common data structures, extracted and expanded
from patterns encountered across LeetCode and CodeWars practice.

| File | Structure | Notes |
|------|-----------|-------|
| `binary_search.py` | Binary Search | Iterative; handles insert-position edge cases |
| `binary_tree.py` | Binary Tree | Node definition + DFS/BFS traversals |
| `linked_list.py` | Singly Linked List | Node, insert, delete, reverse, detect cycle |
| `stack.py` | Stack | Array-backed; used in valid parentheses, clear digits |
| `binary_conversions.py` | Binary ↔ Decimal | Manual bit-manipulation conversions |
