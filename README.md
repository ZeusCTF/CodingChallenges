# Algorithm & Systems Programming Practice

A personal repository of programming practice problems spanning systems programming in C, Python algorithms, and data structures. Built while preparing for software engineering interviews and coursework at UC Berkeley.

## Structure

| Folder | Description |
|--------|-------------|
| [`BerkeleySelfPrep/`](./BerkeleySelfPrep/) | C systems programming exercises: I/O, concurrency, pointers, memory |
| [`LeetCode/`](./LeetCode/) | LeetCode solutions in Python and C |
| [`CodeWars/`](./CodeWars/) | CodeWars kata solutions in Python |
| [`DataStructures/`](./DataStructures/) | Standalone data structure implementations |
